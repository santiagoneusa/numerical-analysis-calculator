1. Expresiones Matemáticas

Al ingresar expresiones matemáticas en la aplicación, es crucial utilizar la sintaxis correcta para que las funciones se interpreten adecuadamente. A continuación, encontrarás una lista detallada de funciones matemáticas y operadores que puedes usar, junto con ejemplos de cómo aplicarlos.

Nombre Sintaxis Ejemplo Descripción
Adición + x + y Suma dos números.
Sustracción - x - y Resta un número de otro.
Multiplicación * x * y Multiplica dos números.
División / x / y Divide un número entre otro.
Exponenciación ** o ^ x**2 o x^2 Eleva un número a una potencia.
Logaritmo Natural log(x) log(e) Calcula el logaritmo natural de x (base e).
Logaritmo Base 10 log10(x) log10(100) Calcula el logaritmo de x con base 10.
Exponencial exp(x) o e^x exp(2) o e^2 Calcula e elevado a la potencia x.
Raíz Cuadrada sqrt(x) sqrt(16) Calcula la raíz cuadrada de x.
Seno sin(x) sin(pi/2) Calcula el seno de x (en radianes).
Coseno cos(x) cos(0) Calcula el coseno de x (en radianes).
Tangente tan(x) tan(pi/4) Calcula la tangente de x (en radianes).
Valor Absoluto abs(x) abs(-5) Devuelve el valor absoluto de x.
Constantes pi, e sin(pi), exp(1) Constantes matemáticas π (≈ 3.1416) y e (≈ 2.7183).

Notas Importantes:

  • Utiliza ** o ^ para la exponenciación. Ambos son aceptados.
  • Asegúrate de que todos los paréntesis estén correctamente cerrados.
  • Las funciones trigonométricas asumen que el valor está en radianes.
  • Para la multiplicación, siempre usa el símbolo * (por ejemplo, escribe 2*x en lugar de 2x).
  • Usa letras minúsculas para variables y funciones.
  • Los espacios son ignorados pero pueden mejorar la legibilidad.

2. Ingreso de Datos

Dependiendo del método que utilices (ecuaciones no lineales, ecuaciones lineales o ecuaciones diferenciales), necesitarás ingresar datos en formatos específicos. A continuación, encontrarás instrucciones detalladas y ejemplos para cada tipo de método.

2.1 Métodos de Ecuaciones No Lineales

Al resolver ecuaciones no lineales, generalmente deberás proporcionar la función, conjeturas iniciales, tolerancias y límites de iteraciones.

Parámetro Descripción Tipo de Dato Ejemplo
Función f(x) La función matemática de la cual deseas encontrar la raíz. Cadena de texto "x**3 - x - 2"
Conjetura Inicial x0 Tu punto de partida para métodos iterativos. Número Real 1.5
Segunda Conjetura Inicial x1 (si es necesario) Utilizada en métodos como Secante o Bisección. Número Real 2.0
Tolerancia El margen de error aceptable para la solución. Número Real 1e-5
Número Máximo de Iteraciones El límite de iteraciones a realizar. Entero 100
Tipo de Error Elige entre error relativo o absoluto para los criterios de parada. Cadena de texto "relative" o "absolute"

Ejemplo de Entrada:

  • Función: "x**3 - x - 2"
  • Conjetura Inicial x0: 1.5
  • Tolerancia: 1e-5
  • Número Máximo de Iteraciones: 100
  • Tipo de Error: "relative"

2.2 Métodos de Ecuaciones Lineales

Para resolver sistemas de ecuaciones lineales, debes ingresar la matriz de coeficientes, el vector de términos independientes, conjeturas iniciales (para métodos iterativos) y tolerancias.

Parámetro Descripción Tipo de Dato Ejemplo
Matriz de Coeficientes A Una matriz cuadrada que representa los coeficientes de las variables. Cadena de texto (Notación de matriz) "[[4, -1], [-1, 3]]"
Vector de Términos Independientes b Un vector que representa los términos independientes en las ecuaciones. Cadena de texto (Notación de vector) "[15, 10]"
Conjetura Inicial x0 Valores iniciales para métodos iterativos. Cadena de texto (Notación de vector) "[0, 0]"
Tolerancia El margen de error aceptable para la solución. Número Real 1e-5
Número Máximo de Iteraciones El límite de iteraciones a realizar. Entero 100

Formato de Entrada para Matrices:

Al ingresar matrices y vectores, puedes utilizar los siguientes formatos:

  • Separa los elementos con espacios o comas.
  • Separa las filas con punto y coma o nuevas líneas.
  • Encierra toda la matriz o vector entre corchetes.

Ejemplos:

  • "[4 -1; -1 3]"
  • "[[4, -1], [-1, 3]]"
  • "[15; 10]" para vectores.

2.3 Métodos de Ecuaciones Diferenciales

Al resolver ecuaciones diferenciales numéricamente, debes proporcionar la ecuación diferencial, condiciones iniciales, rango y tamaño de paso.

Parámetro Descripción Tipo de Dato Ejemplo
Ecuación Diferencial dy/dx = f(x, y) La ecuación diferencial a resolver. Cadena de texto "dy/dx = x + y"
Condición Inicial (x0, y0) El punto de inicio de la solución. Tupla de Números Reales (0, 1)
Rango [a, b] El intervalo en el cual resolver la ecuación. Tupla de Números Reales (0, 5)
Tamaño de Paso h El incremento entre valores de x en la solución. Número Real 0.1

Ejemplo de Entrada:

  • Ecuación Diferencial: "dy/dx = x + y"
  • Condición Inicial: (0, 1)
  • Rango: (0, 5)
  • Tamaño de Paso: 0.1

2.4 Consejos Generales para el Ingreso de Datos

  • Verifica tus entradas en busca de errores de tipeo o sintaxis.
  • Asegúrate de que todas las funciones y variables estén definidas.
  • Utiliza unidades y escalas consistentes para todos los valores numéricos.
  • Si ocurre un error, lee cuidadosamente el mensaje para entender el problema.
  • Consulta la documentación del método para conocer requisitos específicos de entrada.
